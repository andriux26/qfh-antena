Detalles de diseño y construcción de una antena helicoidal

Se presentó la necesidad de la construcción de una antena con polarización circular, y estamos previendo el uso de varias antenas mas de este tipo para el seguimento de satélites polares, motivos suficientes para recopilación de información y publicación...

Ya que la antena helicoidal es de facil construcción, y muy tolerante a variaciones de medidas, es un elemento ideal para experimentación.

Según la literatura, el mínimo número de vuelta de una antena helicoidal varia de 3 a 5. Recién después de 3 vueltas se logra una circularidad aceptable.

Aquí hay una página con imágenes de una helicoide de 3 vueltas que fue construida para un sistema de recepción en la banda de 1690 MHz (satélites geoestacionarios para meteorología). Estas imágenes NO son fotos sino son generados por un programa de visualización en 3D ( POV-Ray).
